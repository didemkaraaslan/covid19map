Covid 19 vaka sayısı , iyileşen sayısı, ölüm sayısı gibi verilerin ülke bazında grafikler ile gösterildiği bir proje. 
Kullanılan Teknolojiler : ReactJS - Redux - MongoDB - NodeJS - react-chartjs-2
